# ApiTest

# 基本功能 
* 项目管理
* 测试环境管理
* 测试用例管理
* 测试集管理
* Jenkins集成

# 技术栈

  >  后台

   * SpringBoot
   * Mybatis
   * redis
   * rest-assured
  
> 前端
   
   * vue2
   * vuex
   * vue-router
   * axios
   * iview
