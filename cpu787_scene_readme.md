  项目说明  
 
     项目采用Spring、SpringMVC、Mybatis实现H5场景制作平台； 
     项目基于Maven进行管理，采用Mysql和Redis实现数据存储 
     db文件夹下为数据库文件 
 
  项目特点  
 
     H5场景的制作：包括添加文字、上传图片、添加背景、添加音乐、文字和图片可以随意移动和修改大小； 
     积分机制：包括积分兑换场景、用户DIY场景上传平台赚取积分； 
     会员机制：会员每月限定次数兑换积分模板； 
     素材丰富：平台内置音乐爬虫； 
     定时机制：每月1号凌晨定时爬去素材、重置会员信息。
