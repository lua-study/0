# TnCMS内容管理系统

#### 介绍
TnCMS原创轻内容管理系统，基于ThinkPHP6+layUI开发。

#### 软件架构

前端：LayUI

后端：ThinkPHP6


#### 功能特点

1.权限管理

2.用户管理

3.文章管理

4.微信公众号登陆

5.前端用户中心

6.用户投稿公众号推送通知

7.微信小程序

8.敬请期待
