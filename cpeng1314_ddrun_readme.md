## 项目介绍
本项目基于Thinkjs3.0+Vue 开发的一套跑腿下单接单系统。
##### 主要功能
地址解析、配送路程计算、物品重量价格计算、分销、微信支付下单、积分抵扣、优惠券、配送系统(抢单模式/平均分配订单)
##### 更新预告
- 1.根据跑男位置就近派单
- 2.设置每日优惠券赠送
- 3.接入公众号服务号更多功能
