# 飞机大战

#### 项目介绍
使用python开发的简单小游戏飞机大战，网上也有很多源码但是多数都不是面向对象的，
本项目采用面向对象编写，逻辑清晰简单明了。适合刚入门学习Python的同学参考！

#### 软件架构
src 下是源码
PlaneMain.py是程序运行入口，右键运行
GameConfig.py是配置文件
PlaneGameVC.py是游戏主页面

Resources 是程序资源文件

#### 安装教程
1.需要安装pygame框架
2.使用PyCharm打开项目即可

#### 使用说明
PlaneMain.py是程序运行入口，右键运行
