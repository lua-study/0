javabb博客

#### 软件架构
软件架构说明 
基本架构：SpringBoot+Mybatis+通用Mapper  
前台框架：Layui

#### 安装教程

1. 先导入doc/db_bmsboot.sql到数据库
2. ide导入项目，修改application-dev.yml中的数据库信息
3. 运行就OJBK了
4. 在浏览器运行localhost   初始账号密码：`javabb/javabb`
